### 구현 대상
- 로그인
  - 행위
    - 유저가 로그인을 한다.
      - 로그인 API 요청을 한다.
    - 유저가 로그아웃을 한다.
      - 로그아웃 API 요청을 한다.
  - 엔티티
    - 유저
  - API
    - auth

- 회원가입
  - 행위
    - 유저가 회원가입을 한다.
  - 엔티티
    - 유저
  - API
    - signup
- 상품 등록
  - 행위
    - 유저가 상품을 등록한다.
  - 엔티티
    - 유저
    - 상품
  - API
    - createProduct

- 상품 수정
  - 행위
    - 유저가 상품을 수정한다.
  - 엔티티
    - 유저
    - 상품
  - API
    - updateProduct

- 상품 리스트
  - 행위
    - 상품을 조회한다.
  - 엔티티
    - 상품
  - API
    - getProducts


### 의존성
- 상품은 유저에 의해 생성되고 수정된다.
